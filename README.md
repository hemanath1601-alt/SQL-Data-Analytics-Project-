# SQL Data Analytics Project

## Project Overview

This project demonstrates data analysis using MySQL on an E-Commerce Orders dataset. The project covers database creation, table creation, data import, and SQL queries to generate business insights.

## Technologies Used

- MySQL Server 8.0
- MySQL Workbench
- Microsoft Excel

## Dataset

- E-Commerce Orders Dataset
- 1200 Records
- 14 Columns

## Features

- Database Creation
- Table Creation
- CSV Data Import
- SQL Queries
- Sales Analysis
- Product Analysis
- Payment Method Analysis
- Order Status Analysis

## SQL Queries

- Display all records
- Count total orders
- Total sales
- Average order value
- Maximum order value
- Minimum order value
- Orders by status
- Product-wise sales
- Payment method analysis
- Referral source analysis
